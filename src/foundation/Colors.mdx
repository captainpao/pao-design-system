## Usage

Colors are available as CSS custom properties and can be used throughout your components:

```css
.my-component {
  background-color: var(--pao-color-primary);
  color: var(--pao-color-primary-text);
  border-color: var(--pao-border-color);
}
```

## Accessibility

All colors in this system meet WCAG 2.1 contrast requirements for text readability. When using colors, ensure sufficient contrast ratios between text and background colors.
