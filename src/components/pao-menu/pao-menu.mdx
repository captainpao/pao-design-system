# Menu

A dropdown menu that pairs a trigger element with a list of selectable `pao-menu-item` children. Supports keyboard navigation (ArrowDown/ArrowUp to move focus, Escape to close) and closes on outside click.

## Events

- `paoOpenChange` on `pao-menu`: Fired when the menu opens or closes. `event.detail = { open: boolean }`
- `paoSelect` on `pao-menu`: Fired when a child item is selected (re-emitted from the child). `event.detail = { value: string }`
- `paoSelect` on `pao-menu-item`: Fired when the item is clicked (not disabled). `event.detail = { value: string }`

## Usage

```html
<pao-menu>
  <pao-button slot="trigger" variant="secondary">Actions ▾</pao-button>
  <pao-menu-item value="edit">Edit</pao-menu-item>
  <pao-menu-item value="duplicate">Duplicate</pao-menu-item>
  <pao-menu-item value="delete" disabled>Delete</pao-menu-item>
</pao-menu>

<script>
  document.querySelector('pao-menu').addEventListener('paoSelect', (e) => {
    console.log('Selected:', e.detail.value);
  });
</script>
```
