# Modal

A dialog overlay that displays content on top of the page. Supports multiple sizes, dismissible backdrop and escape key behavior, and slotted content.

## Events

- `paoOpen`: Fired when the modal is opened via `showModal()`.
- `paoClose`: Fired when the modal is closed. `event.detail = { source: 'escape' | 'backdrop' | 'close-button' }`

## Methods

- `showModal()`: Opens the modal and dispatches `paoOpen`.
- `close()`: Closes the modal and dispatches `paoClose`.

## Usage

The modal uses a slot for its body content. Control visibility with the `open` property or the `showModal()`/`close()` methods. By default, pressing Escape or clicking the backdrop dismisses the modal — set `dismissible="false"` to disable this.
