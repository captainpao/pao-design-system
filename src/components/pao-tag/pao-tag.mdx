# Tag

A compact label element for categorizing, filtering, or highlighting items. Supports multiple variants, two sizes, and an optional remove button.

## Events

- `paoRemove`: Fired when the remove button is clicked (only when `removable` is `true`). No detail payload.

## Usage

```html
<!-- Basic tag -->
<pao-tag>Label</pao-tag>

<!-- With variant -->
<pao-tag variant="success">Active</pao-tag>

<!-- Small removable tag -->
<pao-tag variant="primary" size="sm" removable>React</pao-tag>
```
