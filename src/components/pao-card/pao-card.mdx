# Card

A content container with optional header, body, and footer zones. Purely presentational — no events.

## Slots

<table>
  <thead>
    <tr>
      <th>Slot</th>
      <th>Description</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td><code>header</code></td>
      <td>Card title area — rendered above the body with a bottom border. Hidden when empty.</td>
    </tr>
    <tr>
      <td><em>(default)</em></td>
      <td>Card body content</td>
    </tr>
    <tr>
      <td><code>footer</code></td>
      <td>Card action area — rendered below the body with a top border. Hidden when empty.</td>
    </tr>
  </tbody>
</table>

## Usage

```html
<pao-card shadow>
  <span slot="header">Card Title</span>
  <p>Card body content goes here.</p>
  <div slot="footer">Footer actions</div>
</pao-card>
```
