# Skeleton

A loading placeholder that mimics the shape of content while it loads. Supports text lines, circles, and rectangles with an optional shimmer animation.

## Slots

This component has no slots — it is a self-contained visual placeholder.

## Usage

```html
<!-- Text line placeholder -->
<pao-skeleton variant="text"></pao-skeleton>

<!-- Circle avatar placeholder -->
<pao-skeleton variant="circle" height="48px"></pao-skeleton>

<!-- Image / card placeholder -->
<pao-skeleton variant="rect" height="200px"></pao-skeleton>

<!-- Static (no animation) -->
<pao-skeleton variant="text" ?animated="${false}"></pao-skeleton>

<!-- Card layout placeholder -->
<div style="display: flex; flex-direction: column; gap: 0.75rem; max-width: 320px;">
  <pao-skeleton variant="rect" height="160px"></pao-skeleton>
  <div style="display: flex; gap: 0.75rem; align-items: center;">
    <pao-skeleton variant="circle" height="40px"></pao-skeleton>
    <div style="flex: 1; display: flex; flex-direction: column; gap: 0.5rem;">
      <pao-skeleton variant="text" width="60%"></pao-skeleton>
      <pao-skeleton variant="text" width="40%"></pao-skeleton>
    </div>
  </div>
  <pao-skeleton variant="text"></pao-skeleton>
  <pao-skeleton variant="text"></pao-skeleton>
  <pao-skeleton variant="text" width="75%"></pao-skeleton>
</div>
```
