# Breadcrumb

A navigation landmark that shows the user's location within a hierarchy of pages. Consumer markup places links and the current-page indicator directly in the default slot; the component injects separators via CSS `::slotted`.

## Events

This component emits no custom events.

## Usage

```html
<!-- Basic 3-level breadcrumb -->
<pao-breadcrumb>
  <a href="/">Home</a>
  <a href="/products">Products</a>
  <span aria-current="page">Widgets</span>
</pao-breadcrumb>

<!-- Custom separator -->
<pao-breadcrumb separator="›">
  <a href="/">Home</a>
  <a href="/docs">Docs</a>
  <span aria-current="page">Getting Started</span>
</pao-breadcrumb>
```
