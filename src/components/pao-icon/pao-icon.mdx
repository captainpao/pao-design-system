## Usage Examples

```html
<!-- Basic Icon -->
<pao-icon name="check"></pao-icon>

<!-- Different Sizes -->
<pao-icon name="user" size="sm"></pao-icon>
<pao-icon name="user" size="md"></pao-icon>
<pao-icon name="user" size="lg"></pao-icon>

<!-- Different Colors -->
<pao-icon name="times" color="danger"></pao-icon>
<pao-icon name="check" color="success"></pao-icon>
```
