# Toast

A brief notification that appears to provide feedback or status. Supports multiple severity variants and optional auto-dismiss.

## Events

- `paoDismiss`: Fired when the toast is dismissed. `event.detail = { variant: string }`

## Usage

Control visibility with the `open` property. For floating toasts, position the host element with CSS (`position: fixed`, etc.).
