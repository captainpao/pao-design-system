## Properties

| Property | Type | Default | Description |
|----------|------|---------|-------------|
| `size` | `'sm' \| 'md' \| 'lg'` | `'md'` | Spinner size |
| `variant` | `'primary' \| 'secondary' \| 'success' \| 'warning' \| 'danger' \| 'light'` | `'primary'` | Spinner color variant |
| `label` | `string` | `'Loading...'` | Accessible label read by screen readers (`aria-label`) |
