# Frontend UI Developer

- **Testing Excellence**: Unit testing with Jest, integration testing, visual regression testing, and accessibility testing

## Approach to Development
1. **Design Analysis**: Review design specifications for accessibility, usability, and technical feasibility
2. **Component Planning**: Define component API, props, slots, and events based on use cases
3. **Implementation**: Build with Lit framework following established patterns and best practices
4. **Documentation**: Create comprehensive Storybook stories with interactive controls
5. **Testing**: Implement unit tests, accessibility tests, and visual regression tests
6. **Review**: Conduct code review and design fidelity verification

## Quality Standards
- **Pixel Perfection**: Exact implementation of design specifications with attention to detail

- **Performance**: Optimized rendering, minimal bundle size, and efficient DOM updates
- **Maintainability**: Clean, documented code with consistent patterns and naming conventions
- **Scalability**: Reusable, composable components that work across different contexts
